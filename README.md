## 🚀 About The Project

This is a state-of-the-art interactive developer portfolio designed to stand out. Moving away from generic developer templates, this project utilizes a custom **"Aerospace Orange"** design system that evokes precision engineering and high-end software development ("BUILDS THINGS THAT FLY").

The experience is heavily focused on micro-interactions, cinematic storytelling, and smooth physics.

### ✨ Key Features:
* **Interactive Starfield:** A custom Three.js / React Three Fiber raycaster engine that causes 2,000 3D background stars to dodge the user's cursor in real-time.
* **Cinematic Audio:** An Easter egg ambient audio player featuring Hans Zimmer's "First Step" from the *Interstellar* Original Motion Picture Soundtrack, seamlessly integrated with smooth volume transitions.
* **Magnetic Physics:** Navigation links and call-to-action buttons feature magnetic, physics-based hover states that pull toward the cursor.
* **Custom Routing:** Framer Motion implementation wrapped around Next.js App Router for elegant page slide and fade transitions.
* **Aerospace UI Theme:** Custom dark mode palette utilizing Deep Void Black (`#050507`), Slate White (`#E2E8F0`), and Safety Orange (`#FF3B00`).

## 💻 Getting Started

### Prerequisites

* Node.js 18.x or later
* npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AstaadDahiya/portfolio-2026.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
